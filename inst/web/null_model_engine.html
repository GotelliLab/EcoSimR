Run null model

Usage

null_model_engine(speciesData, algo, metric, nReps = 1000, saveSeed = FALSE, algoOpts = list(), metricOpts = list(), type = NULL, suppressProg = FALSE)

Arguments

speciesData
a dataframe of data that will work with metrics and algorithms.
algo
the algorithm to use
metric
the metric used to caluclate the null model
nReps
the number of replicates to run the null model.
saveSeed
Should the existing random seed be saved to make the model reproducible?
algoOpts
a list containing options for a supplied alogrithm
metricOpts
a list containing options for a supplied metric
type
The type of null model you are running, if it's meant to integrate with existing null models the type should be "size","niche!","cooc", or leave it NULL if you it's not meant to be compatible with existing null models and a generic null model will be returned.
suppressProg
a parameter to suppress the progress bar. Mostly this is just used for creating documentation with knitr

Description

This drives the null models for all the different kinds of null models that can be run. It is the underlying engine.
