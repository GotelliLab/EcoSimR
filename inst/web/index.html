EcoSimR

EcoSimR is a software package to fit null models to ecological data. Currently three different types of models are supported: co-occurrence, niche overlap, and size ratio. The software comes with existing alogrithms and metrics, but individual users can also integrate their own algorithms, or use the null model engine to create new null models.
