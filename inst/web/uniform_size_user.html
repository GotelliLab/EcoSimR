User defined size limits

Usage

uniform_size_user(speciesData = runif(n = 20), userLow = 0.9 * min(speciesData), userHigh = 1.1 * max(speciesData))

Description

Function to randomize uniformly body sizes within user-defined limits. Note that all n species are randomized in this algorithm
