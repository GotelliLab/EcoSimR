Co-Occurrence Model Plot function

Usage

"plot"(x, type = "hist", ...)

Arguments

x
the null model to plot
type
the type of null plot to make. See details for more information
...
Other variables to be passed on to base plotting

Description

Plot co-occurrence null model object.

Details

the valid types for size are "hist" to show a histogram and "size" to show a sample size null model.
