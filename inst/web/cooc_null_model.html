Co-Occurrence Null model

Usage

cooc_null_model(speciesData, algo = "simFast", metric = "C.Score", nReps = 1000, rowNames = TRUE, randomSeed = 0, burnin = NA, algoOpts = list(), metricOpts = list())

Arguments

speciesData
a dataframe
algo
the algorithm to use, must be "sim1", "sim2", "sim3", "sim4", "sim5", "sim6", "sim7", "sim8", "sim9", "sim10"
metric
the metric used to caluclate the null model: choices are "Species.Combo", "Checker", "C.Score", "C.Score.var", "C.Score.skew", "V.Ratio"; default is "C.Score"
nReps
the number of replicates to run the null model.
rowNames
Does your dataframe have row names? If yes, they are stripped, otherwise FALSE for data that has no row names
randomSeed
Choose a seed to start your random number. 0 will choose a random seed, otherwise set the seed with any integer.
burnin
The number of burnin iterations to use with the simFast algorithm

Description

Create a Co-Occurrence null model
